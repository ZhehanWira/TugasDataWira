# Tugas E-Commerce

**Laporan Praktikum**
  1. Business Question:
     - Siapa pelanggan terbaik yang layak mendapatkan voucher loyalitas berdasarkan analisis RFM?


       
  2. Data Wrangling:
     - Mengubah Order_Date ke format datetime.
     - Mengecek missing value dan duplikasi data.
     - Menyesuaikan tipe data numerik (Total_Sales, Ad_Budget).
     - Membersihkan data agar siap dianalisis.


       
  3. Insights:
     - Pelanggan dengan skor RFM tinggi (lebih dari 13) layak mendapat voucher loyalitas.
     - Beberapa kategori produk lebih efisien menghasilkan penjualan meski budget iklannya kecil.
     - Budget iklan yang lebih tinggi cenderung menghasilkan penjualan lebih besar.


       
  4. Reccomendation:
     - Fokuskan iklan pada kategori paling efisien.
     - Evaluasi kategori yang budget iklannya besar tetapi penjualannya rendah.
